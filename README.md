# Quora Mini Project (REST API Practice)

This is a mini project where I practiced CRUD operations using REST API concepts.  
This project is a Quora-like platform built with Node.js and Express.

## Features
- Create / Read / Update / Delete Posts
- REST API following proper HTTP methods

## Tech Stack
- Node.js
- Express.js
- Nodemon (for development auto-reload)

## How To Run Locally

1. Clone the repo
   ```bash
   git clone https://github.com/ABDULLAH-SWR/REST-API-MINI-PROJECT.git
2. Install Dependencies
   ```
   npm install
   npm intall express
   npm install method-override
   npm install nodemon
3. Run index.js using nodemon in the working directory
   ```
   nodemon index.js
4. Browse the API endpoint
   ```
   localhost:8080/posts
